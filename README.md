# Healthcare Risk Stratification Dashboard

## 📊 Project Overview
This project presents an Excel-based data analytics framework to evaluate clinical and metabolic risk factors associated with chronic disease, particularly diabetes.

## 🎯 Objectives
- Perform data cleaning and feature engineering on clinical datasets  
- Develop a risk scoring model  
- Analyze relationships between biomarkers and risk levels  
- Build an interactive dashboard for insights  

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- XLOOKUP, IF formulas  
- Data Visualization  
- Correlation Analysis  

## 📈 Key Features
- Risk classification (High / Medium / Low)  
- KPI summary with clinical indicators  
- Correlation analysis of variables  
- Interactive dashboard with slicers  

## 🔍 Key Insights
- HbA1c and glucose are the strongest predictors of risk  
- High-risk individuals show ~18× higher diabetes prevalence  
- Cardiovascular factors (SBP, cholesterol) contribute to risk  
- Age increases risk significantly
